\chapter{Experiments}\label{chap:experiments}

This chapter provides empirical validation of the theoretical claims developed in Chapters~\ref{chap:centralized}--\ref{chap:marl}. We first describe core implementation details that specialize the MMATPG as a finite, formula-indexed simulation environment with a GPT-based MARL training pipeline. We then present five experiments, each validating a core theoretical result or illuminating the learning dynamics:

\begin{enumerate}
    \item \textbf{Centralized scheduling.} RLS achieves the predicted $(2-1/m)$ approximation ratio. (\S\ref{sec:exp-centralized})
    \item \textbf{Publication externality and market internalization.} Credit-based rewards produce unbounded PoA; the contract market eliminates the externality. (\S\ref{sec:exp-externality-market})
    \item \textbf{MARL convergence and equilibrium quality.} PSRO+MAPPO converges to approximate CCE with bounded PoA. (\S\ref{sec:exp-marl})
    \item \textbf{Ablation.} Each framework component contributes measurably. (\S\ref{sec:exp-ablation})
    \item \textbf{Scaling.} Theoretical bounds hold across instance sizes and agent counts. (\S\ref{sec:exp-scale})
    % \item \textbf{Qualitative trajectory analysis.} Inspection of learned agent behavior. (\S\ref{sec:exp-qualitative})
\end{enumerate}


\subsection{Implementation Details}\label{sec:implementation}

We implement a finite specialization of the MMATPG that preserves the core dynamics while making simulation and learning tractable. We describe the key design choices here; full implementation details (hyperparameters, training schedules, observation encoding) are deferred to Appendix~\ref{app:implementation}.

\paragraph{Formula Universe and Latent Graph}

The formula space $\mathcal{F}$ is a finite set of $2n$ integer-indexed formulas, organized as $n$ theorem pairs. For each theorem $k\in\{0,\ldots,n{-}1\}$, formula $k$ and formula $k+n$ are negation partners. A latent truth assignment $T:\{0,\ldots,n{-}1\}\to\{0,1\}$ selects one formula per pair as true; false formulas can be attempted but the proof kernel assigns them zero success probability.

Each instance is equipped with: (i) a difficulty map $d:\mathcal{F}_\top\to(0,1)$ governing base proof rates; (ii) a dependency DAG over true formulas giving the unique latent $\delta^*$; and (iii) a utility graph $G_U = (\mathcal{F}_\top, E_U, w)$ with edge weights $w(\psi,\phi)=1$ for hard dependencies and $w(\psi,\phi)\in(0,1)$ for soft support.

\paragraph{Sequent Compression}

To prevent exponential blowup in the action space, we collapse sequents to formula targets under a canonical local context. When agent $\alpha$ targets formula $\phi$, the sequent is automatically constructed as $[\mathcal{R}^\alpha_t \proves \phi]$, where $\mathcal{R}^\alpha_t$ is the set of formulas $\alpha$ has currently resolved. This induces a bijection between concrete formulas and concrete sequents, and ensures that all resolved sequents are automatically fully resolved (since antecedents contain only already-resolved formulas).


\paragraph{Proof Kernel}

Proof outcomes are Bernoulli with deterministic dependencies. For agent $\alpha$ attempting $\phi$ with effective cumulative budget $\tau_{\mathrm{eff}}$ and prior accumulated effort $\tau_{\mathrm{prev}}$:
\[
p^\alpha_{\mathrm{proof}}(\phi, \tau_{\mathrm{eff}}) = g(\phi)\cdot\Big(1 - \exp\!\big(-\rho^\alpha(\phi)\cdot(\tau_{\mathrm{eff}}^\kappa - \tau_{\mathrm{prev}}^\kappa)\big)\Big),
\]
where $g(\phi) = \mathbf{1}[T(\phi)=1]\cdot\mathbf{1}[\delta^*(\phi)\subseteq\mathcal{R}^\alpha_t]$ is a feasibility gate, $\kappa\in(0,1]$ governs diminishing returns, and $\rho^\alpha(\phi) = e^{-\lambda_{\mathrm{diff}}\cdot d(\phi)}\cdot\big(\rho_0^\alpha + \rho_1^\alpha\cdot\log(1 + \sum_{\psi} w(\psi,\phi)^\alpha)\big)$ combines difficulty modulation with utility-weighted support.

\paragraph{Other Kernels, Market, and Training}

The conjecture kernel shares the conditional Weibull shape as the proof kernel with a rate driven by local opportunity mass over ghost formulas. The query model is an oracle with persistent per-formula noise. The market is the collateralized bilateral contract market from Chapter~\ref{sec:decentralized}, with an exogenous bounty agent seeding long offers at initialization. Training uses PPO with GAE advantages, entropy regularization, AdamW optimization, and a curriculum over five difficulty levels ($n \in \{4,8,16,32,64\}$ theorem pairs).


\subsection{Experimental Setup}\label{sec:exp-setup}

\paragraph{Instance parameters.}
Unless otherwise noted: $m=2$ agents, synthetic DAG instances with depth $d=3$, branching factor $b=2$, $\kappa=0.9$, oracle query model, $\ell = 0.1$, horizon $H=50$.

\paragraph{Baselines.}
\begin{itemize}
    \item \textbf{OPT}: Exact centralized optimum via dynamic programming (feasible for $n\le 16$).
    \item \textbf{RLS}: Reactive List Scheduling with shortest-expected-processing-time (SEPT) priority.
    \item \textbf{Greedy credit}: Each agent greedily maximizes credit-based reward; no market, no publication of byproducts.
    \item \textbf{Scripted market}: Accept all available long bounties, prove highest-value target, publish immediately.
    \item \textbf{MARL}: Full framework (MAPPO + population-based training + curriculum).
\end{itemize}

\paragraph{Metrics.}
All metrics averaged over $N_{\mathrm{eval}} = 50$ rollouts with $95\%$ confidence intervals.
\begin{itemize}
    \item $\mathrm{SC}$: Expected makespan. \quad $\mathrm{PoA} := \mathrm{SC}/\mathrm{OPT}$.
    \item $\varepsilon\text{-exploit}$: Maximum unilateral improvement via MAPPO fine-tuning of a deviator. Measures distance to CCE.
    \item $\mathrm{Pub\%}$: Fraction of resolved formulas that are published.
    \item $\mathrm{Resolve\%}$: Fraction of target formulas fully resolved by horizon.
\end{itemize}

\textbf{Findings.} As $m$ increases, PoA grows toward the bound (from $1.29$ to $1.61$) while absolute makespan falls sharply (from $35.12$ to $19.34$). Training time scales roughly as $O(m^{1.8})$ over this range, driven by the multi-head attention over agent-specific market tokens.

The right panel of Figure~\ref{fig:e5-agents} makes the practical case: at $m=8$, MARL achieves SC $= 19.34$ vs.\ centralized RLS at approximately $18.5$ (estimated, since OPT is not tractable at $n=8$, $m=8$). The decentralization penalty in absolute time is small---the market mechanism effectively coordinates eight independent agents to within $\approx 5\%$ of the centralized optimum.
